# OMR Epikrisis Control Parser 2023-2025

Контрольный парсер выписных эпикризов ОМР1 за 2023, 2024 и 2025 годы.

## Что ищет

Рядом с exe должны лежать три папки:

- `ОМР 1 2025`
- `ОМР1 2024`
- `ОМР1 2023`

Внутри каждой папки обязательны:

- `ВМП`
- `ДМС`
- `ОМС`
- `ПМУ`

Парсер рекурсивно читает только `.doc`, `.docx`, `.rtf`.

По умолчанию дополнительно берутся только файлы с годом создания `2023`, `2024` или `2025`.

## Выход

По умолчанию результат создается на рабочем столе:

`Выписки_2023_2025_CONTROL`

Структура:

```text
Выписки_2023_2025_CONTROL/
  Подтвержденные/
  review/
    likely_discharge/
    weak_match/
    duplicate_same_patient_same_date/
    read_error/
  logs/
    manifest.xlsx
    manifest.csv
    run_log.txt
    summary.txt
```

## Важные правила

- Сетевая папка используется только на чтение.
- Все копии и логи пишутся в локальную папку результата.
- После завершения программа сама закрывается и не ждет Enter.
- После успешного завершения упакованный Windows exe удаляет сам себя.
- Windows shutdown не вызывается; компьютер уснет по системным настройкам.
- Если произошла фатальная ошибка, failed staging-папка сохраняется для диагностики.

## Запуск

Обычный запуск двойным кликом:

```text
epikrisis_control_2023_2025.exe
```

Полезные параметры:

```text
--pause-at-end             ждать Enter в конце
--keep-exe                 не удалять exe после успешного завершения
--ignore-creation-year     не фильтровать файлы по году создания
--dry-run                  только сканировать и писать logs, без копирования
--source "D:\path"         указать папку, где лежат годовые папки
--output "C:\path\out"     указать папку результата
```

## Сборка локально

На Windows с Python:

```bat
build_exe.bat
```
